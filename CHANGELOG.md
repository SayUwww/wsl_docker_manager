# 更新日志

## v1.1.0 - 2026-07-09

### 新增
- 新增 Remote 连接模式，使用 Rust 内置 SSH 连接远程 Linux 服务器并管理远程 Docker。
- 新增远程服务器配置管理，支持保存、选择、删除、测试远程服务器配置。
- 新增统一设置弹窗，将连接模式、刷新时间、语言、主题、关闭按钮行为、开机自启、远程服务器配置集中管理。
- 新增开机自启选项，自启时支持隐藏到托盘。
- 容器页新增快捷筛选，支持按分组和状态过滤容器。
- 容器、镜像、网络、数据卷读取命令新增加载状态，读取中禁用重复刷新并显示加载提示。
- 连接模式切换、Remote 模式下切换远程服务器、保存当前远程服务器配置时，新增全局加载蒙版。

### 优化
- 切换连接模式后会自动刷新 Docker 状态、系统资源、容器、镜像、网络、数据卷数据。
- 优化设置弹窗 UI，关闭按钮移动到右上角，内容区独立滚动。
- 优化 Toast 层级，打开设置弹窗或全局加载蒙版时仍能看到错误提示。
- 优化系统托盘右键菜单，移除“启动全部容器/暂停全部容器”，改为显示主窗口、隐藏到托盘、刷新数据、打开设置、退出。
- 仪表盘中运行中容器和已停止容器最多显示 3 个，避免列表过长影响概览。
- 执行日志详情改为更外层弹窗展示，避免在日志区域内显示不完整。
- 执行日志中的命令展示为实际 Docker 指令文本，便于排查失败原因。
- 远程 Docker 命令支持自定义 Docker socket，并通过 `DOCKER_HOST` 注入执行环境。

### 修复
- 修复最小化到托盘按钮无效的问题。
- 修复 WSL 命令输出乱码问题。
- 修复 `dev.ps1` 生成的 WSL 环境变量中 Windows 路径带空格导致 shell 解析失败的问题。
- 修复远程 `docker ps -a` 在部分 Docker 版本缺少 `ImageId` 字段时解析失败的问题。
- 修复容器页 sticky 区域顶部透出下方内容的问题。
- 修复切换连接模式时局部加载图标位置偏移的问题，改为全局加载蒙版。

## v1.0.1 - 2026-07-09

### 新增
- 仪表盘执行日志支持点击查看详情。
- 执行失败后可在日志详情中查看错误信息并重新执行。
- 用户配置支持持久化保存，包括刷新时间、语言、连接模式、主题。
- 点击窗口关闭按钮时可选择最小化到托盘或直接关闭，并支持记住选择。
- 容器、镜像、网络、数据卷页面新增手动刷新按钮。

### 优化
- 容器页 sticky 区域覆盖标题、搜索、批量操作和表格头。
- 镜像页和数据卷页取消 sticky。
- 执行日志详情弹窗改为全局弹窗。

### 修复
- 修复执行日志乱码问题。
- 修复关闭按钮最小化到托盘相关权限缺失问题。
