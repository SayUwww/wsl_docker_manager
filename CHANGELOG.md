# 更新日志

## v1.0.1 - 2026-07-09

### 新增
- 仪表盘执行日志支持点开，在独立的全局弹窗中查看详情。
- 执行日志详情支持查看命令参数、运行输出、耗时、状态，并可重新执行原操作。
- 执行日志现在显示面向 Docker 的实际指令文本，例如 `docker ps -a`、`docker start <id>`，不再只显示内部 Tauri 方法名。
- 用户配置支持本地持久化，包括刷新间隔、语言、连接模式、主题和关闭按钮行为。
- 点击窗口关闭按钮时，可选择最小化到托盘或直接退出应用，并可记住选择。
- 容器、镜像、网络、数据卷页面新增手动刷新按钮。
- 容器页滚动时会固定显示标题、搜索控件、批量操作和表格头。

### 变更
- 应用启动时会先将持久化的连接模式同步到 Tauri 后端，再开始轮询 Docker 数据。
- 执行日志详情弹窗改为渲染到文档根节点，避免被仪表盘日志区域裁剪。
- WSL 命令输出在 Windows 下增加 GBK 解码回退，改善中文 Windows/WSL 错误信息的可读性。
- 最小化到托盘逻辑会检查窗口是否已隐藏；如果隐藏失败，则回退为窗口最小化。

### 修复
- 修复执行日志详情弹窗显示不完整或被裁剪的问题。
- 修复执行日志中 WSL/Docker 错误信息乱码的问题。
- 修复容器页固定区域顶部存在缝隙、滚动内容会透出的显示问题。
- 修复最小化到托盘、窗口隐藏和关闭行为所需的 Tauri 权限缺失问题。
